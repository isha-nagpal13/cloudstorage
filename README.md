# 🚀 AI Cloud Storage Platform

A modern, AI-powered cloud storage web application with intelligent file management, smart search, and automated insights.

---

## ✨ Overview

This project provides a clean and scalable cloud storage solution enhanced with AI-driven features such as smart recommendations, duplicate detection, and automatic tagging.

Designed with a focus on usability, performance, and modern UI/UX principles.

---

## 🧠 Key Features

* 🔐 **Authentication System**
  Secure login and signup functionality

* 🤖 **AI Recommendations**
  Suggests relevant files based on user behavior

* 🔍 **Smart Search**
  Natural language-based file search

* 📊 **Storage Analytics**
  Visual insights into storage usage

* 🧬 **Duplicate Detection**
  Identifies and manages similar files

* 🏷️ **Auto Tagging**
  Automatically categorizes uploaded files

* 📁 **File Management**
  Upload, download, and delete files efficiently

* 📊 **Dashboard**
  Clean and intuitive user interface

## 🚀 Getting Started

### 1. Install dependencies

```bash
npm install
```

### 2. Run the frontend

```bash
npm run dev
```

Frontend runs on:

```
http://localhost:3000
```

---

## 🔗 Backend Services

Ensure the following services are running:

* Backend API → http://localhost:5000
* AI Engine → http://localhost:5001
